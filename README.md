# Vim Dotfiles

Checkout my dank vim setup.

## Installation

```
$ git clone git@github.com:conornewton/vim-dotfiles
$ cd vim-dotfiles && ./install.sh
```

Afterwards to install vim plugins

```
$ vim
:PlugInstall
```
To fully set up coc.nvim for auto completion

```
$ vim
:call coc#util#install())
```
